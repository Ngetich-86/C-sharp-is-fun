# setting up your project
This is a guide to setting up your project.

## Step 1
Console App
```bash
dotnet new console -n MyProject
```
Web App
```bash
dotnet new web -n MyProject
```
Web API
```bash
dotnet new webapi -n MyProject
```
Blazor WebAssembly
```bash
dotnet new blazorwasm -n MyProject
```
